# Remotion Video: Gemini 10s Graphic Ad

This folder contains a complete Remotion composition that matches your 10-second script.

## Scenes implemented

- **0:00–0:03** Hook: floating icons organize into a clean grid with the headline **“Smarter. Faster. Better.”**
- **0:03–0:06** Solution: the grid transitions to a glowing Gemini-style burst mark with the line **“Google Gemini. The Ultimate AI.”**
- **0:06–0:10** Action & outro: rapid cards for writing/coding/creating plus a search bar prompt and cycling CTA text.

## Run locally

```bash
cd remotion
npm install
npm run start
```

## Render MP4

```bash
cd remotion
npm run render
```

The output is written to:

```text
remotion/out/gemini-ad.mp4
```
